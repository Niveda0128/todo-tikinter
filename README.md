# To-Do List GUI App (Tkinter)

## 📌 Project Name
To-Do List GUI Application

## 📖 Description
This is a simple desktop To-Do List application built using Python Tkinter.  
It helps users to add, complete, and delete tasks easily.

## ⚙️ Features
- Add new tasks
- Mark tasks as completed ✔️
- Delete tasks 🗑️
- Simple and user-friendly GUI

## 🖥️ How to Run
1. Install Python
2. Open terminal or VS Code
3. Run the file:
   python todo.py
